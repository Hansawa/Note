# 技术栈

## 前后端分离

### 后端

1. java企业级应用框架：SpringBoot
2. ORM框架：MyBatis
3. 数据库：MySQL
4. 数据库连接池：Druid

### 前端

1. nodejs版本管理器：nvm
2. javascript运行时：nodejs（基于Chrome V8引擎）
   1. 软件包管理器：pnpm
   2. 软件包管理器：npm
3. Vue3-Vite2
   1. 路由：vue-router
   2. 状态管理：vuex
   3. css预处理器：scss
   4. 异步请求组件：axios
   5. markdown编辑器：v-md-editor
   6. markdown编辑器升级组件：codemirror
   7. 样式框架
