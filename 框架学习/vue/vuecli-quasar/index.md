什么是 quasar？一个 vue 组件库

使用 vue-cli 进行安装

```shell
1. # npm 安装vue-cli
npm i -g @vue/cli

2. # vue-cli 创建 vue 项目
vue create <name>

3. 进入该项目

4. quasar 的安装
vue add quasar

5. 各种插件的安装
# vue提供的前端路由
npm i vue-router@next -S

# vue提供的状态管理工具vuex
npm i vuex@next -S

# 基于 promise 的网络请求库axios
npm i axios@next -S

# css预处理器scss的库
npm i sass -D

# scss-loader
npm i sass -D

```
